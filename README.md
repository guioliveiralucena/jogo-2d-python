
# Jogo 2D

Descrição breve do seu projeto Pytho.

## Requisitos

- Python 3.8+
- pip

## Instalação

```bash
git clone https://github.com/seu-usuario/jogo-2d.git
cd jogo-2d
pip install -r requirements.txt
```

## Uso

```bash
python main.py
```
